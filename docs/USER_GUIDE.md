# Пользовательский гайд MelodicaTrainer

MelodicaTrainer помогает тренироваться на мелодике: показывает клавиатуру, слушает микрофон, проверяет попадание в ноты, воспроизводит партитуры и показывает их как падающие ноты в режиме `Tabs`.

Приложение состоит из четырех основных разделов:

| Раздел | Для чего нужен |
| --- | --- |
| `Melodica` | Проверить высоту сыгранной ноты и увидеть ее на клавиатуре. |
| `Tabs` | Загрузить MusicXML/MXL/Guitar Pro, включить playback и играть по Note Highway. |
| `Practice` | Тренировать гаммы, отдельные ноты, аккордовые звуки и 12-bar blues. |
| `Circle` | Изучать круг квинт, лады, гаммы и трезвучия. |

## Общие элементы

Верхнее меню открывает разделы `Circle`, `Melodica`, `Tabs` и `Practice`.

Переключатель `A-B-C` меняет отображение нот между буквенной нотацией и сольфеджио. Расчеты внутри приложения от этого не меняются, меняются только подписи.

Приложение использует микрофон в разделах `Melodica`, `Practice` и `Tabs`. При первом запуске прослушивания браузер попросит разрешение. Если доступ запрещен, приложение покажет ошибку, а распознавание нот работать не будет.

Часть настроек сохраняется в браузере: выбранный диапазон мелодики, tempo, SoundFont, режим отображения нот, закрепление панелей и загруженный файл. Если поведение кажется странным после старых экспериментов, обычно помогает сменить настройку вручную или перезагрузить страницу.

## Melodica

Раздел `Melodica` нужен для быстрой проверки сыгранных нот.

1. Выберите `Melodica range`.
2. Нажмите `Start listening`.
3. Разрешите доступ к микрофону, если браузер спросит.
4. Сыграйте ноту на мелодике.
5. Смотрите на блок `Detected note` и подсветку клавиши.

Основные элементы:

| Элемент | Что делает |
| --- | --- |
| `Melodica range` | Выбирает размер клавиатуры: 25, 27, 32, 37 или 44 клавиши. Вариант на 44 клавиши соответствует диапазону Hammond 44 (C3-G6). |
| `Start listening` / `Stop listening` | Включает или выключает микрофон. |
| `Detected note` | Показывает распознанную ноту. |
| `cents` | Показывает, насколько нота выше или ниже точной высоты. |
| `clarity` | Показывает уверенность распознавания сигнала. |

Если линия настройки на клавише смещена, нота звучит неточно. Значение около `0 cents` означает хорошее попадание. Положительное значение значит, что нота звучит выше нужной высоты; отрицательное - ниже.

## Tabs

Раздел `Tabs` - основной режим для игры по партитурам. По умолчанию открывается score `Aloutte`.

### Загрузка файла

Нажмите `Load XML/GP` и выберите файл. Поддерживаются:

- `.xml`
- `.musicxml`
- `.mxl`
- `.gp`
- `.gp3`
- `.gp4`
- `.gp5`
- `.gpx`

Для MusicXML/MXL приложение показывает ноты через OpenSheetMusicDisplay. Для Guitar Pro используется alphaTab. В обоих случаях Note Highway строится из разобранных событий playback.

Важно: приложение работает с первой релевантной партией/первым staff для MusicXML и с выбранным `GP Track` для Guitar Pro.

### Левая панель

Левая панель содержит настройки файла и звука.

| Элемент | Что делает |
| --- | --- |
| `Melodica Range` | Выбирает диапазон, под который строятся подписи и Note Highway. |
| `SoundFont` | Выбирает банк звуков для playback. |
| `Instrument` | Выбирает инструмент внутри SoundFont, если он доступен. |
| `GP Track` | Выбирает дорожку Guitar Pro. Видно только для GP-файлов. |
| `Load XML/GP` | Загружает новый файл. |
| `XML` | Скачивает транспонированный MusicXML. Доступно для MusicXML. |
| `Text` | Скачивает текстовый список нот мелодики. Доступно для MusicXML. |

### Playback

Верхняя панель на странице `Tabs` показывает controls playback:

| Элемент | Что делает |
| --- | --- |
| `Play` | Запускает воспроизведение. |
| `Pause` | Ставит воспроизведение на паузу. |
| `Resume` | Продолжает с текущего места. |
| `Restart` | Возвращает песню к началу. |
| `-5 BPM` / `+5 BPM` | Уменьшает или увеличивает tempo на 5 BPM. |
| tempo slider | Меняет tempo вручную. |
| `Hits`, `Miss`, `Streak`, `ACC` | Показывает статистику попаданий. |

Также внизу верхнего меню есть progress bar песни.

### Шорткаты клавиатуры

Шорткаты работают в разделе `Tabs`, если фокус не находится в поле ввода, select или textarea.

| Шорткат | Что делает |
| --- | --- |
| `Space` | Запускает playback или ставит его на паузу. |
| `Escape` | Перезапускает score с начала, как `Restart`. |
| `+` или `=` | Увеличивает tempo на 5 BPM. |
| `-` или `_` | Уменьшает tempo на 5 BPM. |

### Note Highway

Центральная область показывает Note Highway:

- падающие блоки - ноты, которые нужно сыграть;
- горизонтальная линия - момент попадания;
- подпись `Press play`, `Listening` или распознанная нота показывает состояние микрофона;
- `Clarity` показывает качество текущего распознавания;
- цветные вспышки показывают попадания.

Нота считается засчитанной, когда сыгранная высота совпадает с целевой нотой в момент прохождения target line. Если микрофон недоступен или сигнал слишком тихий, hits не будут засчитываться.

### Правая панель

Правая панель управляет отображением и транспонированием.

| Элемент | Что делает |
| --- | --- |
| `Transpose` | Сдвигает ноты на указанное число полутонов. |
| `Reset transpose` | Возвращает транспонирование к `0`. |
| `Show Note Names` | Показывает или скрывает названия нот на падающих блоках. |
| `Fingering Guide` | Выбирает подсказки пальцев: `None`, `Numbers on notes`, `Virtual hand`. |
| `Study Mode` | Останавливает продвижение на целевой ноте, пока она не сыграна. |
| `Optimize` | Ищет вариант транспонирования, который лучше помещается в выбранный диапазон мелодики. |

Если рядом с `Optimize` показано число вариантов, повторные нажатия перебирают найденные варианты.

### Панели score/settings

На широком экране у `Tabs` есть три панели:

- левая панель настроек;
- верхняя score-панель;
- правая панель transpose/display.

Кнопки `Pin panel`, `Unpin panel`, `Pin score`, `Unpin score` закрепляют или скрывают панели. Если панель не закреплена, ее можно вызвать наведением мыши на край экрана: слева, справа или сверху.

Верхнее меню тоже можно закреплять и скрывать кнопкой `Pin menu` / `Unpin menu`. Если меню не закреплено, наведите мышь на верхний край окна, чтобы временно показать его.

### Действия мышью

В центральной области `Tabs`:

| Действие | Что делает |
| --- | --- |
| Левый клик по Note Highway | Запускает playback или ставит его на паузу. |
| Колесо мыши вверх | Увеличивает tempo на 5 BPM. |
| Колесо мыши вниз | Уменьшает tempo на 5 BPM. |
| Правый клик / context menu по Note Highway | Закрепляет все панели, если они скрыты, или открепляет все панели, если они уже закреплены. |
| Наведение на левый край | Показывает левую панель настроек, если она не закреплена. |
| Наведение на правый край | Показывает правую панель transpose/display, если она не закреплена. |
| Наведение на верхний край score | Показывает верхнюю score-панель, если она не закреплена. |
| Наведение на верхний край окна | Показывает верхнее меню, если оно не закреплено. |

## Practice

Раздел `Practice` нужен для целевой тренировки нот и гамм.

1. Выберите `Melodica range`.
2. Выберите `Tonic`.
3. Выберите `Scale`.
4. Выберите режим тренировки.
5. Нажмите `Start listening`.
6. Играйте целевые ноты и следите за статусом `Hit` / `Waiting`.

Режимы:

| Режим | Как использовать |
| --- | --- |
| `Explore` | Свободно изучайте клавиатуру: подсвечиваются ноты выбранной гаммы. |
| `Notes` | Приложение выбирает целевые ноты из выбранной гаммы. |
| `Scale` | Цели идут по порядку гаммы. |
| `Chord tones` | Тренируются ноты аккорда от выбранной тоники. |
| `12-bar` | Показывает 12 тактов блюза и аккордовые звуки текущего такта. |

Кнопка `Next target` переключает цель. В режимах с микрофоном попадание засчитывается, когда сыграна правильная MIDI-нота и интонация достаточно близка к центру.

## Circle

Раздел `Circle` - теория и визуальная ориентация по тональностям.

Основные действия:

- нажмите ноту на внешнем круге, чтобы выбрать root;
- выберите mode в ряду кнопок;
- выберите scale в следующем ряду;
- смотрите выбранную гамму, scale degrees и triads в нижней панели.

Обозначения:

| Элемент | Значение |
| --- | --- |
| cyan border | Tonic of selected mode, стартовая нота выбранного лада. |
| numbers inside circle | Scale degrees. |
| green | Scale note. |
| yellow | Major triad. |
| blue | Minor triad. |
| red | Diminished triad. |
| dark/gray | Нет трезвучия или качество не определено. |

`Circle` не использует микрофон и не зависит от выбранного диапазона мелодики.

## Troubleshooting

### Браузер не дал доступ к микрофону

Проверьте разрешения сайта в браузере и разрешите microphone access. После изменения разрешений перезагрузите страницу и снова нажмите `Start listening` или `Play`.

### Нота не определяется

Проверьте, что микрофон включен, сигнал достаточно громкий и выбранный диапазон мелодики включает сыгранную ноту. Слишком тихий или шумный сигнал может давать низкий `clarity`.

### Файл не загружается

Проверьте расширение файла. Для `Tabs` поддерживаются MusicXML/MXL и Guitar Pro форматы. Если файл поврежден или не содержит партитуру, приложение покажет сообщение об ошибке.

### Файл слишком большой

Приложение ограничивает размер MusicXML/MXL файлов. Используйте более компактный экспорт или упростите файл в редакторе партитур.

### Не слышно playback

Проверьте громкость системы и вкладки браузера. Нажмите `Play` после загрузки score. Если выбранный SoundFont или Instrument не звучит ожидаемо, попробуйте другой `SoundFont` или `Instrument`.

### GP track не тот

Для Guitar Pro выберите нужную дорожку в `GP Track`. После смены дорожки Note Highway и score должны перестроиться под выбранную партию.

### Ноты не попадают в диапазон мелодики

В `Tabs` попробуйте:

1. Изменить `Melodica Range`.
2. Нажать `Optimize`.
3. Вручную изменить `Transpose`.
4. Нажать `Reset transpose`, если нужно вернуться к исходному варианту.

### Статистика не засчитывает попадания

Проверьте микрофон, `clarity`, громкость сигнала и выбранный диапазон. В `Study Mode` playback будет ждать правильную ноту, поэтому отсутствие продвижения обычно означает, что нужная нота еще не распознана.
